I have monitor more than 40 experiments in this year, Here is one most common reason experiments fails.

Calling of Experiments too Early

Most of time, it is most common discuss we had with stack holder, as soon as it looks like experiment reach significance level even if it doesn’t have statistical power. This is way power analysis upfront comes very useful. Ability to show that it is still not time is critical tool for data scientist how have.

Difference between assignment and applied

Some time Experiment start time and implimentation time can be different, it is critical know if there is difference. Example, if experimentation treatment need to be applied by human,for example, sales person talking to client. In this case, treatment is created long before applying it. Another Example, if you are using api call to get experimentation, sometimes, api call will fail or take longer than SLA, now, treatment suppose to be applied but it was not.
